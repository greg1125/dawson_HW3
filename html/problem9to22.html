9. JavaScript is a dynamically typed language. What does that mean?
JavaScript is dynamically typed, meaning variables do not have a fixed data type and can hold
different types of values at different times

10. let is block-scoped and var is not. What does that mean?
let is only accessible inside the block {} where it is declared,
whereas var is accessible anywhere within the function it was declared in,
even outside {} blocks.
